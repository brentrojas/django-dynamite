1. Abstract Dynamic Models¶

class dynamite.dymodels.AbstractSchema(*args, **kwargs)[source]¶

Bases: django.db.models.base.Model

This abstract model represent the S in a schema - table - field pattern.

You should inherit from this class when you want to start a dynamical model system.

import dynamite

class Survey(dynamite.models.AbstractSchema):
    class Meta:
        verbose_name = 'Survey'
        verbose_name_plural = 'Surveys'

Note

To activate this schema remember to register it in yourapp.models with:

dynamite.site.register(Survey)
